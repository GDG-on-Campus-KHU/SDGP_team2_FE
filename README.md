## 📌 Overview

> **ECO-BEAN** is an eco-friendly collection network platform  
> that connects **cafes and users** to reduce the careless disposal of coffee grounds.

- 🗺️ **Search collection points via map**
- 🧑‍🍳 **Register and manage coffee grounds (for cafes)**
- 🤖 **AI-powered suggestions for ground usage**
- 📊 **Auto-generated environmental impact reports**

---

## 🎯 Goals

- ♻️ **Promote recycling of coffee grounds**
- 👥 **Automate matching between users and cafes**
- 🧠 **Provide AI-powered solutions via Gemini API**
- 🌍 **Encourage eco-conscious behavior**

---

## 🧩 Key Features

### 🗺️ 1. Home (Map-based collection point search)
- Map display based on current location
- Clickable markers → show popups with details
- Filters for cafes/public/enterprise

### 🔐 2. Login / Signup
- Social login via Kakao or Google
- Choose user type (General user / Cafe owner)

### 🧑‍🍳 3. Cafe Dashboard
- Register bean information
- Add grounds and accept/reject collection requests
- Manage cafe profile

### 👤 4. User MyPage
- View collection request history
- Check request status (Pending / Accepted / Completed)
- Access contribution reports

### 🤖 5. AI Solution *(Coming Soon)*
- Personalized suggestions for reuse (e.g., fertilizer, scrub)
- Analyze environmental contributions using Gemini API

### 📊 6. Environmental Report
- Visualize carbon reduction and recycling score
- Compare data between users and cafes

> 🧾 **Example Report**  
> “This month, you recycled 5kg of coffee grounds,  
> reducing approximately **4.25kg of CO₂** —  
> equivalent to a full day's breath of an adult!”

### 🛍️ 7. Upcycling Market / Community *(Coming Soon)*
- Share or trade upcycled coffee products
- Community board
- (Later feature: 1:1 Chat)
